个人和工作的github账号分离

git@github.com:macrozheng/mall.git -> git@personal:macrozheng/mall.git 个人
git@github.com:macrozheng/mall.git -> git@work:macrozheng/mall.git 工作

实现原理，是给每一个github 账号生成了公钥，在enter file which to save the key 时 命名公钥文件
